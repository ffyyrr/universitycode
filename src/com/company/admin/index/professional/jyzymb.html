计算机科学与技术（专业代码：080901H）
2020-05-19 15:35     (浏览量：)

一、专业介绍：

本专业拥有硕士学位授予权，培养掌握知识、具备能力和素质的计算机专业工程技术人才。本专业已形成特色鲜明的实践教学实训体系，Oracle、软通动力、H3C等国内外著名IT企业先后在本专业建立了实训基地，良好的实践教学条件为每位学生提供了大量实践训练机会。毕业后能在计算机相关行业,特别是互联网应用领域从事设计、开发、测试、运维、管理等工作，也可进一步深造，从事计算科学研究，攻读硕士博士学位。英国阿尔斯特大学90%的计算机科学方面的研究项目具有世界领先水平。


二、培养目标：

本专业培养具有开阔的国际视野和良好的交流沟通能力，具有扎实的计算机软、硬件理论知识；掌握计算机相关领域所需的工程知识；具有利用现代工具对相关复杂工程问题进行分析、研究、解决和管理能力；具有良好的职业素养、团队精神、创新意识、沟通交流与自我提升能力；能够在IT及相关行业进行项目研究与开发、企业运营管理和技术服务的高素质国际化应用型人才。


三、主要课程：

1 离散数学 Discrete Mathematics

2 数据结构与算法 Data Structures and Algorithms

3 面向对象程序设计 Object-Oriented Programming–C++/Java

4 计算机组成原理与系统结构 Computer Composition Principle and System Structure

5 计算机网络 Computer Network

6 操作系统 Operating Systems

7 数据库原理及应用 Database Principle and Application

8 算法分析与设计 Algorithm Analysis and Design


四、授予学位(双学位）：

1.陕西科技大学工学学士学位

2.阿尔斯特大学理学学士学位
